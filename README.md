# Reflowable.me - PDF to EPUB Converter

A full-stack application that converts fixed-layout PDFs into reflowable EPUB files.

## Tech Stack
- **Frontend:** React, Vite, Tailwind CSS, Axios
- **Backend:** FastAPI, PyMuPDF (Fitz), EbookLib

## Setup
1. **Backend:** - `cd backend`
   - `pip install -r requirements.txt`
   - `uvicorn main:app --reload`
2. **Frontend:**
   - `cd frontend`
   - `npm install`
   - `npm run dev`
